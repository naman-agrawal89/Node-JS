# PanCakeSwap TX Server
Tool to decode pancake swap router swap transcations input parameter and query data from last few blocks.

## Usage

```
$ npm i 
$ npm start
```

# API Doc

API endpoints: 

* GET /decode
* GET /lastblocks
* GET /lasttx

#### 1. GET /decode
